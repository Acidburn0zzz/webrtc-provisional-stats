RTCAudioReceiverStats dictionary non-standardized members 

partial dictionary RTCAudioReceiverStats {
      unsigned long long jitterBufferFlushes;
      unsigned long long delayedPacketOutageSamples;
};
jitterBufferFlushes of type unsigned long long

Counts the total number of times the jitter buffer reaches its maximum capacity and gets flushed.

delayedPacketOutageSamples of type unsigned long long

Measures the number of audio samples that were concealed due to delayed packets. This gives an indication of how well the jitter buffer's network adaptation is working.
